The program is written in Python 2.7.13. It uses several Windows specific libraries and will not run on other operating systems.
To run the program you need to place the randomizer.py file and cheats.txt file in the same folder.
Navigate to the folder from command prompt, or press Shift + Right Click in the folder and press 'Open PowerShell window here'.
In the command line type the following: python randomizer.py
Note: Your python executable might be named differently (python27 for example), so if you cannot run the program check that. 
It is commonly located in: C:\Program Files\Python27.
